# Patches.com

A modern Vue.js web application for patch management.

## Features

- Beautiful gradient branding (light blue to purple)
- Responsive top navigation
- Side navigation with quick access to key sections
- Modern homepage with hero card and action buttons

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- Vue 3
- Vite
- Modern CSS with gradients and animations
